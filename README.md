# NeuroStack Mobile SPA

A multi-file, single-page app with:

- Extreme 3D depth neumorphism.
- Local note taking.
- Gemini helper page.
- Prompt creation page.
- No framework and no build step.

## Files

- `index.html`
- `styles.css`
- `app.js`
- `gemini.js`

## Notes

- Notes are stored in browser localStorage.
- Gemini runs client-side from the browser.
- For a private or production AI setup, move the API call behind your own server later.
